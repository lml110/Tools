# searchControl

> 行内搜索筛选器

Author: liumouliang

## Props

| Prop name | Description  | Type            | Values | Default |
| --------- | ------------ | --------------- | ------ | ------- |
| value     | 操作数据     | object          | -      |         |
| data      | 筛选数据     | array           | -      | []      |
| search    | 搜索按钮配置 | boolean\|object | -      | true    |
| reset     | 重置按钮配置 | boolean\|object | -      | true    |
| size      | 尺寸         | string          | -      | 'small' |
| recover   | 重置状态     | boolean         | -      | false   |
