## loadResource(url:, type) ⇒ <code>url</code>
<p>加载资源</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| url: | <code>String</code> | <p>文件远程地址</p> |
| type | <code>String</code> | <p>css || js || pre</p> |

**Example**  
```javascript
loadResource(url,'js')
```
