**Returns**: <code>Number</code> - <p>时间戳</p>  
**Date**: 2021-11-11  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| date | <code>Date</code> |  |

// 客户端显示时间
