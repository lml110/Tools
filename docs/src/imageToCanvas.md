## imageToCanvas(src, options) ⇒ <code>Promise</code>
<p>图片链接转canvas</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| src | <code>String</code> | <p>图片链接</p> |
| options | <code>Object</code> | <p>配置项</p> |

**Example**  
```javascript
imageToCanvas(blob,{
    width: 210,
    height: 270,
}).then
```
