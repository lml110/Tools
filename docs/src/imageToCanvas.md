## imageToCanvas(src, isBase64) ⇒ <code>Promise</code>
<p>图片链接转canvas</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| src | <code>String</code> | <p>图片链接</p> |
| isBase64 | <code>Boolean</code> | <p>可选-是否转base64</p> |

**Example**  
```javascript
imageToCanvas(src)
```
