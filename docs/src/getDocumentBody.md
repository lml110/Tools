## getDocumentBody([propertyName]) ⇒ <code>any</code>
<p>兼容获取body元素</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| [propertyName] | <code>String</code> | <p>属性名|可为空</p> |

**Example**  
```javascript
getDocumentBody() //html
console.log(getDocumentBody('clientWidth')); // 938
```
