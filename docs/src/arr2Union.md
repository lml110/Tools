## arr2Union(arr1, arr2) ⇒ <code>Array</code>
<p>获取数组的并集</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| arr1 | <code>Array</code> | <p>源数据</p> |
| arr2 | <code>Array</code> | <p>目标数据</p> |

**Example**  
```javascript
console.log(arr2Union([1,2,4,[2,5]],[2,5,[3,4]],true)); //[2, 5, 4]
```
