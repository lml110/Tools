## getIdCardLastChar(cid) ⇒ <code>Number</code>
<p>获取身份证号第18位校验码</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| cid | <code>value</code> | <p>17或18位均可</p> |

**Example**  
```javascript
console.log(getIdCardLastChar(331081198609201016));	// 8
```
