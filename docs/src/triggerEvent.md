## triggerEvent(elem, name)
<p>兼容性click元素触发</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| elem | <code>Element</code> |  |
| name | <code>String</code> | <p>事件名|默认：click</p> |

**Example**  
```javascript
triggerEvent(a); // a自动触发点击事件
//== elem.click()
```
