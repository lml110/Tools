## \_regAngle(str, regStr) ⇒ <code>String</code>
<p>替换英文字符为全角字符</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| str | <code>String</code> |  |
| regStr | <code>String</code> | <p>正则 默认字典中全部字符</p> |

**Example**  
```javascript
// 根据匹配出来的字符，字典内的替换，字典外的默认为空_regAngle(str)
```
