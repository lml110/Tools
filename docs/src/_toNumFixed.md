## \_toNumFixed(f, [fixed]) ⇒ <code>Number</code>
<p>转数字</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| f | <code>any</code> | <p>表达式</p> |
| [fixed] | <code>Number</code> | <p>可填-最大位数</p> |

**Example**  
```javascript
console.log(_toNumFixed(.2-.3),_toFixed(.2-.3)); == -0.1 '-0.10'
```
