## getFileExtName(std) ⇒ <code>String</code>
<p>获取后缀名并转换小写</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| std | <code>str</code> | <p>文件地址</p> |

**Example**  
```javascript
getFileExtName('04/2019110405043483.JPG') == 04/2019110405043483.jpg
```
