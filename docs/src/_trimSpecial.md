## \_trimSpecial(str, regStr) ⇒ <code>String</code>
<p>去除特殊字符</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| str | <code>String</code> |  |
| regStr | <code>Reg</code> | <p>正则 | 默认全部</p> |

**Example**  
```javascript
// 正则内特殊：\\+ | \\[ | \\]// 行装：&+\\/↵\r\n\t_trimSpecial(str)
```
