## \_float(f, [digit]) ⇒ <code>Number</code>
<p>浮点数计算</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| f | <code>any</code> | <p>表达式</p> |
| [digit] | <code>Number</code> | <p>位数</p> |

**Example**  
```javascript
console.log(_float(2.312*.3)); //0.69
console.log(_float(1.235, 0)); //1
```
