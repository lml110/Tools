## colorHex(sRgb) ⇒ <code>String</code>
<p>RGB颜色转换为16进制</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| sRgb | <code>String</code> | <p>RGB值</p> |

**Example**  
```javascript
console.log(colorHex('rgb(23, 245, 56)'));   // #17f538console.log(colorHex('#fff'));   // #ffffff
```
