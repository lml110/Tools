## numberPad(source, len) ⇒ <code>String</code>
<p>补充差值，过滤负数|字符</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| source | <code>Number</code> \| <code>String</code> |  |
| len | <code>Number</code> | <p>可选-长度|默认2</p> |

**Example**  
```javascript
console.log(numberPad(2));  //'02'
console.log(numberPad(-2)); //'-02'
```
