## numberRandom(max, min) ⇒ <code>Number</code>
<p>生成随机数</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| max | <code>Number</code> | <p>可填-最大位数|默认9</p> |
| min | <code>Number</code> | <p>可填-最小位数|默认为0</p> |

**Example**  
```javascript
// min <= val <= max// 默认一个数值 10以内numberRandom()
```
