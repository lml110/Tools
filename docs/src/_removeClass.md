## removeClass(element, className)
<p>从指定元素中移除指定的类名。</p>

**Date**: 2017-11-23  
**Author**: 移除class类  

| Param | Type | Description |
| --- | --- | --- |
| element | <code>HTMLElement</code> | <p>需要移除类名的DOM元素。</p> |
| className | <code>string</code> | <p>需要移除的类名。</p> |

**Example**  
```javascript
_removeClass(element, 'lml')
```
