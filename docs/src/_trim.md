## \_trim(str, is_global) ⇒ <code>String</code>
<p>去掉字符串空格</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| str | <code>String</code> | <p>字符串</p> |
| is_global | <code>Boolean</code> | <p>可选 - 是否去除全部空格 || 默认去除首尾空格</p> |

**Example**  
```javascript
console.log(_trim([' dfd ds dd ','ss 是否'],true));
```
