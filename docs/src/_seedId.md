## \_seedId(maxNum) ⇒ <code>Number</code>
<p>生成不重复的唯一值-数值型</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| maxNum | <code>Number</code> | <p>最大长度</p> |

**Example**  
```javascript
# 使用numberRandom需计算最大值
console.log(_seedId(12)); //349842133076
console.log(_seedId(5)); //62890
```
