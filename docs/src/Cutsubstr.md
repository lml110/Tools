## Cutsubstr(str, len, ifChinese)
<p>截取字符串+... 中英文区分</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| str | <code>String</code> |  |
| len | <code>Number</code> | <p>最大长度</p> |
| ifChinese | <code>Boolean</code> | <p>可填 是否计算中文字符| 默认true</p> |

**Example**  
```javascript
Cutsubstr(str, len)
```
