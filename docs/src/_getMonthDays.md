## \_getMonthDays(date, month) ⇒ <code>Number</code>
<p>获取某一年某一月的总天数</p>

**Date**: 2017-11-23  
**Author**: liumouliang  

| Param | Type | Description |
| --- | --- | --- |
| date | <code>Date</code> | <p>默认：获取当前月份</p> |
| month | <code>Number</code> | <p>月份</p> |

**Example**  
```javascript
方式一：$.getMonthDays();
方式二：$.getMonthDays(new Date());
方式三：$.getMonthDays(2013, 12);
```
